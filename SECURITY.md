# Security policy

## Supported status

This repository is a private staging repository and is not presented as production-ready software.

## Reporting

If you identify a security issue in code or documentation, report it privately to the repository owner through a non-public channel.

Do not include secrets, tokens, private keys, or personal data in issue reports or patches.

## Current validation evidence

The repository has explicit evidence for:

- secret/privacy scan passed
- `detect-secrets` passed
- `bandit` passed

There is no verified evidence here that the C projects have been compiled, fuzzed, or runtime-tested.

## Disclosure expectations

Because this repository may contain reconstructed educational systems code, fixes should be reviewed for both security impact and academic-integrity impact before publication.
