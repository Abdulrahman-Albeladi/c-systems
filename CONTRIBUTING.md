# Contributing

This repository is currently a private reconstruction workspace.

## Scope

Contributions should be limited to:

- removing non-portable or assignment-derived material
- improving build clarity after commands are verified
- adding small tests for behavior that is clear from the code
- improving documentation about boundaries, assumptions, and interfaces

## Do not contribute

- copied assignment text
- instructor or autograder tests
- starter-code files with unclear licensing
- grading metadata
- private data or personal identifiers

## Review standard

Changes should be conservative:

- separate verified facts from assumptions
- avoid claiming build success or test success without evidence
- preserve existing attribution and ownership information
- prefer small, reviewable changes over broad rewrites

## Before proposing public release

Confirm that:

- the relevant project was independently reconstructed
- no course-specific material remains
- ownership and licensing are settled
- collaborator attribution and permission are settled if applicable
